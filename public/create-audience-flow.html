LUMOS · Audiences

Create Audience — inside Data Explorer

Clicking + Create audience doesn't open a separate surface — it drops the user into Data Explorer, the app's existing chat-plus-session-rail workspace. The sidebar's Data Explorer lights up, the chat opens a guided brief, and the audience takes shape as a card in the session rail on the right. Selecting it slides in the standard Audience detail panel — the same one used across the product — where the profile builds live and the user refines by talking. Save files it as a reusable object that also lands in the Audiences library. Nothing new to learn; it's the surface people already explore data in.

Hitting Save audience flips the panel to a Saved state (hero pill + top-bar), keeps it in this session's rail and files it under its auto-category in the Audiences library — a persistent, reusable object. The footer swaps to View in library / Open full, and the chat offers the next moves: open the full profile, build a campaign, or create another. The user never left Data Explorer.
